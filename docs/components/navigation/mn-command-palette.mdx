# MnCommandPalette

Keyboard-driven command palette for quick actions and navigation.

## When to use

Use to give power users a fast command and search interface.

## Props

_This component extends native HTML attributes. See source for full type details._


## Example

```tsx
import { MnCommandPalette } from "@/components/maranello"

<MnCommandPalette />
```

## Accessibility

- Implements `aria-current` for active navigation items
- Full keyboard navigation support with arrow keys
