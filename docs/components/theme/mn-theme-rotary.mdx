# MnThemeRotary

Rotary dial for selecting the active color theme with preview swatches.

## When to use

Use to let users browse and switch themes via a physical-dial metaphor.

## Props

_This component extends native HTML attributes. See source for full type details._


## Example

```tsx
import { MnThemeRotary } from "@/components/maranello"

<MnThemeRotary />
```

## Accessibility

- Theme changes are announced via `aria-live`
- All controls meet WCAG 2.2 AA contrast requirements
