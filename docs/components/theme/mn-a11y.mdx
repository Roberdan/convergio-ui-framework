# MnA11y

Accessibility settings panel with contrast, font-size, and motion controls.

## When to use

Use to let users customize accessibility preferences app-wide.

## Props

_This component extends native HTML attributes. See source for full type details._


## Example

```tsx
import { MnA11y } from "@/components/maranello"

<MnA11y />
```

## Accessibility

- Theme changes are announced via `aria-live`
- All controls meet WCAG 2.2 AA contrast requirements
