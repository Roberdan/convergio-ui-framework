# MnSearchDrawer

Full-height search drawer with real-time results and category filters.

## When to use

Use for global search experiences that overlay the main content.

## Props

_This component extends native HTML attributes. See source for full type details._


## Example

```tsx
import { MnSearchDrawer } from "@/components/maranello"

<MnSearchDrawer />
```

## Accessibility

- Form controls are associated with labels via `htmlFor`/`id`
- Validation errors are announced to screen readers
- Supports keyboard navigation between fields
